# MTechSolutions Website

Professional consulting website for MTechSolutions, built with Next.js.

## Live Site

[www.mtechsolutions.ie](https://www.mtechsolutions.ie)

## Development

The Next.js site is located in the `mtechsolutions-site/` directory.

\`\`\`bash
cd mtechsolutions-site
npm install
npm run dev
\`\`\`

Visit http://localhost:3000 to view the site locally.

## Deployment

The site automatically deploys to GitHub Pages via GitHub Actions when changes are pushed to the main branch.
